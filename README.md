# FireProducts

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.0.6.


# Running App!

Install the dependencies and devDependencies and start the server.

```sh
$ cd fireProduct
$ npm install -d
$ ng serve -o
```
